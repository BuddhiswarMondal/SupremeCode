Welcome Into Array World

What is array?

array হল অনেকগুলি data কে একসংগে রাখার মতো container ।

array কিভাবে declair করা হয়?

array লেখা হয় এইভাবে- var array = ["Ram", "Rohim", "Sunil", 20, "true", "kabita"];

আবার var array= new Array; লিখে ও array declair করা যায়।

array method কী?

array কে Javascript এর সাহায্যে control and manupulate করা হয় যার সাহায্যে, তাকে array methode বলে।

array methode গুলি কি কি?

sort(), rverse(), pop(), push(), shift(), unshift(),concat(), join(), slice(), splice(), indexOf(), lastIndexOf(), isArray(), include(), some(), every(), find(), findIndex(), filter(), toString(), valueOf(), fill(), forEach() total 23pc।

array.sort() কী?

array ঘোষনা হওয়ার পর, উক্ত array গুলিকে alphabetically সাজানোকে sort() করা বলে।

যেমনঃ-var array = ["Ram", "Bimal", "Sunil","Chandan" , "Lalita", "kabita"]; একে sort() করলে alphabetically হবে-

Ram, Bimal, Chandan, kabita, Lalita, Sunil

arry.reverse() কী?

array যেভাবে আছে, তাকে পিছন থেকে সাজালে বা উলটো দিক দিয়ে সাজানোকে array.reverse() বলে।
