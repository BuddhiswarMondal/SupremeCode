Q.1) What is BOM or Browser object Model?

A.1) The Browser Object Model provides JavaScript with the ability to interact with the browser itself, rather than just the content of the web page (which is handled by the DOM – Document Object Model).

Q.2) What can we do by BOM?

A.2) With the Browser Object Model (BOM) in JavaScript, you can interact with the browser environment in ways that go beyond manipulating the web page’s content. Here's a breakdown of what you can do with the BOM:

    We Can Control Browser

  • Open and close window.
const newWindow = window.open("https://example.com", "_blank"); newWindow.close();

    We can resize or move window

  • Rsize Or Move Window
    • window.resizeTo(800, 600); window.moveTo(100, 100);
