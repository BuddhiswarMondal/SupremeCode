HTML Basics

HTML (HyperText Markup Language) structures web content. Here's a basic example:

<!DOCTYPE html>
<html>
  <head>
  <h1> Who Are You </h1>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>Welcome to my website!</p>
  </body>
</html>

CSS Basics

CSS (Cascading Style Sheets) styles HTML elements. Example:

body {
  background-color: lightblue;
  font-family: Arial, sans-serif;
}
h1 {
  color: navy;
  text-align: center;
}

JavaScript Basics

JavaScript adds interactivity. Here's a basic script:

<script>
  function greet() {
    alert("Hello, welcome to the site!");
  }
  window.onload = greet;
</script>
